# PLAYERZWZ — AI-Powered Sports Matchmaking & Arena Booking

PLAYERZWZ is an intelligent platform designed to reduce coordination friction when planning casual sports matches and reserving arena slots. By combining location-based filtering with an AI recommendation module, it matches active players based on sport preference, proximity, availability, and an established reliability score.

## 🛠️ Technology Stack
* **Frontend:** Flutter (Dart)
* **Backend:** Flask (Python) REST API 
* **Database & Auth:** Firebase Authentication & Cloud Firestore (NoSQL)
* **AI Engine:** Scikit-learn (History-based sequence ranking & Rule-based fallback)
* **External Integration:** Google Maps/Places API & Firebase Cloud Messaging (FCM)

## 📌 Core Features (Epics Covered)
* **EP-01 & EP-02:** Secure User Administration & Player Profiles
* **EP-03 & EP-04:** Nearby Player Discovery & Custom Game Creation
* **EP-05:** Request-based Arena Slot Booking Verification
* **EP-06:** AI-Assisted Recommendation Engine (Proximity, History, and Reliability signals)
* **EP-07 & EP-08:** Fraud-resistant Post-game Ratings & FCM Push Notifications
